# simple-obd
Simple OBD Scanner made with python-obd package
# Installation
1. Open a Terminal/Command Prompt
<br>
<br>
2. Select your Directory using:
<br>
cd <folder-path>
<br>
<br>
3. Install required packages by typing:
<br>
For **Python 3.x**
<br>
pip3 install colorama obd
<br>
For **Python 2.x**
<br>
pip2 install colorama
<br>
pip2 install obd
<br>
<br>
4. Running
<br>
For **Python 3.x/2.x**
<br>
python3 main.py
<br>
or
<br>
python2 main.py
